# DeepGuard V8.0 EXA-CYBER 🛡️

Industrial-grade Deepfake Detection & Forensic Suite. High-precision tri-modal analysis engine featuring Spatial, Spectral, and Texture probe vectors. 

## 🛡️ Core Capabilities
- **Tri-Modal Fingerprinting**: Concurrent analysis via **Xception-71** (Spatial), **2D-FFT** (Spectral), and **LBP Entropy** (Texture).
- **Forensic PDF Reports**: Automated generation of secure, timestamped case files for every audit.
- **Neural Identity Management**: Secure JWT-based authentication with case-isolated audit ledgers.
- **Exa-Cyber UI**: Professional command-center interface with real-time hardware telemetry and trajectory analytics.

## 🛠️ Tech Stack
- **Backend**: FastAPI (Python), SQLAlchemy, PyTorch, Timm, OpenCV, FPDF2, Scikit-Image.
- **Frontend**: React-Vite (TypeScript), Tailwind CSS v4, Framer Motion, Recharts, Lucide-React.
- **Database**: SQLite (SQLAlchemy ORM).

## 🚀 Quick Execution
1. **Backend**:
   ```bash
   cd backend
   pip install -r requirements.txt
   python main.py
   ```
2. **Frontend**:
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
